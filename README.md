# Library Management System

## Overview

The Library Management System is a console-based Java application developed using Core Java concepts. It allows a librarian to manage books and students, issue and return books, and maintain library records.

This project was built as a beginner-friendly Java project to practice Object-Oriented Programming (OOP), Collections, and File Handling.

---

## Features

* Add a new book
* View all books
* Search a book by Book ID
* Remove a book
* Add a new student
* View all students
* Issue a book to a student
* Return a book
* Maintain issued books for each student

## Classes Used

### Book

Stores information about a book.

Attributes:

* Book ID
* Book Name
* Author
* Availability Status

### Student

Stores information about a student.

Attributes:

* Student ID
* Student Name
* List of Issued Books

### Library

Contains all library operations such as:

* Add Book
* View Book
* Search Book
* Remove Book
* Add Student
* View Student
* Issue Book
* Return Book

### Main

Displays the menu and handles user interaction.

## How to Run

1. Clone or download the project.
2. Open the project in IntelliJ IDEA or Visual Studio Code.
3. Compile all Java files.
4. Run `Main.java`.
5. Use the menu to perform library operations.
